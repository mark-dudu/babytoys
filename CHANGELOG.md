# 更新日志

本项目使用语义化版本，并在发布前把“未发布”内容移动到对应版本。

## 未发布

### 新增

- 增加低刺激进入方式，新配置默认立即黑屏，也可选择短暂显示图片后渐暗。
- 超时睡眠唤醒后继续保持儿童模式，直到家长主动解锁。

### 改进

- 旧版配置升级后保留原有图片渐暗行为，新建预设默认使用立即黑屏。
- 睡眠恢复和显示器变化后重新覆盖全部屏幕并检查输入拦截；恢复失败时完整清理并安全退出。

### 修复

- 主窗口版本号改为读取程序集版本，避免发布后仍显示旧版本号。

### 文档

- 明确 BabyToys 面向婴幼儿家庭，通过降低电脑吸引力来减少乱拍键鼠造成的误操作，而不是强化安全锁屏。
- 将 v1.3 主线收敛为低吸引力体验和睡眠唤醒后的持续保护，并明确暂不加入密码、管理员模式、远程解锁、安装器或自动更新。

## 1.2.0 - 2026-07-18

### 新增

- 设置与儿童模式时序策略的自动化测试。
- Windows CI、单文件发布打包和 SHA-256 校验文件。
- 标签驱动的 GitHub Release 工作流。
- 单实例运行保护。

### 改进

- 设置文件增加结构版本、统一归一化和原子写入。
- 损坏的设置文件会先备份，再恢复安全默认值。
- 来自更高版本的设置文件会进入只读保护，避免旧版本覆盖或降级配置。
- 应用日志限制单文件大小并保留最近 14 个文件。
- 时长、解锁时长、倒计时和睡眠结果决策从 UI/计时器代码中抽离。
- 固定构建 SDK，并使用 NuGet 锁文件保证 CI 与发布还原一致。

## 1.1.0

- 增加托盘常驻、全局快捷键、预设、开机自启、诊断与异常会话恢复。
- 改进长按解锁反馈，并统一应用图标。

## 1.0.0

- 首个源码版本：多显示器遮罩、常规输入拦截、长按解锁和超时睡眠。
