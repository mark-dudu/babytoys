# BabyToys

Windows 儿童模式防误触工具。一键进入儿童模式后，程序会覆盖所有显示器、短暂显示系统壁纸或自定义图片，随后转为黑屏，并尽量拦截常规键盘鼠标输入。倒计时结束后会优先请求系统睡眠。

## 目标边界

第一版目标是防孩子乱拍导致误操作、降低屏幕吸引力、保证家长可恢复。它不是安全锁屏软件，也不承诺拦截 `Ctrl+Alt+Del`、硬件电源键、UAC 安全桌面或所有系统级快捷键。

家长解锁方式：`Ctrl + Alt + U` 长按 3 秒。

## 技术栈

- C# / .NET 10 / WPF
- Windows 10/11
- WPF 原生控件
- Win32 P/Invoke 封装系统能力
- 无第三方依赖

## 运行

需要在 Windows 上安装 .NET 10 SDK。

```powershell
dotnet run
```

发布单文件版本：

```powershell
dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
```

## 当前功能

- 主窗口设置图片来源和时间限制，支持小数分钟；时间会限制在 0.5 到 240 分钟之间。
- 默认使用系统壁纸，也支持自定义图片。
- 进入前有 3 秒确认倒计时，可取消。
- 儿童模式覆盖所有显示器。
- 图片阶段叠加深色遮罩，10 秒后淡入纯黑屏。
- 使用 low-level keyboard hook 和 mouse hook 尽量拦截常规输入。
- 超时后释放钩子并请求系统睡眠；睡眠请求成功后结束儿童模式，睡眠请求失败时保持黑屏并允许解锁。
- 配置保存到 `%LocalAppData%/BabyToys/settings.json`。
- 日志写入 `%LocalAppData%/BabyToys/logs`，只记录进入、退出、超时和错误，不记录具体按键内容。
