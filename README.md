# BabyToys

Windows 儿童模式防误触工具。一键进入儿童模式后，程序会覆盖所有显示器、短暂显示系统壁纸或自定义图片，随后转为黑屏，并尽量拦截常规键盘鼠标输入。倒计时结束后会优先请求系统睡眠。

当前开发版本为 v1.1：在 v1 儿童模式基础上加入托盘常驻、快捷启动、配置预设和开机自启；仍暂不提供安装器、自动更新或安全锁屏承诺。

## 目标边界

第一版目标是防孩子乱拍导致误操作、降低屏幕吸引力、保留家长可恢复路径。它不是安全锁屏软件，也不承诺拦截 `Ctrl+Alt+Del`、硬件电源键、UAC 安全桌面或所有系统级快捷键。

家长解锁方式：`Ctrl + Alt + U` 长按 3 秒。

进入儿童模式前有 3 秒确认倒计时，可点击取消或按 `Esc` 取消。

## 技术栈

- C# / .NET 10 / WPF
- Windows 10/11
- WPF 原生控件
- Win32 P/Invoke 封装系统能力
- 无第三方依赖

## 运行

需要在 Windows 上安装 .NET 10 SDK。

```powershell
dotnet build
dotnet run
```

发布单文件版本：

```powershell
dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
```

输出文件位于 `bin/Release/net10.0-windows10.0.17763.0/win-x64/publish/`。

## 当前功能

- 主窗口设置图片来源和时间限制，支持小数分钟；时间会限制在 0.5 到 240 分钟之间。
- 默认使用系统壁纸，也支持自定义图片。
- 进入前有 3 秒确认倒计时，可点击取消或按 `Esc` 取消。
- 儿童模式覆盖所有显示器。
- 图片阶段叠加深色遮罩，10 秒后淡入纯黑屏。
- 使用 low-level keyboard hook 和 mouse hook 尽量拦截常规输入。
- 超时后释放钩子并请求系统睡眠；睡眠请求成功后结束儿童模式，睡眠请求失败时保持黑屏并允许解锁。
- 配置保存到 `%LocalAppData%/BabyToys/settings.json`。
- 日志写入 `%LocalAppData%/BabyToys/logs`，只记录进入、退出、超时和错误，不记录具体按键内容。
- 支持保存、应用和删除多组儿童模式预设。
- 支持 `Ctrl + Alt + B` 全局快捷键启动，仍会经过 3 秒确认倒计时。
- 关闭主窗口后继续在系统托盘运行，可从托盘打开、启动儿童模式或安全退出。
- 可选择登录 Windows 后自动启动到托盘。

## 已知限制

- 不能阻止 `Ctrl + Alt + Del`、UAC 安全桌面、硬件电源键或所有系统级快捷键。
- 输入拦截依赖 Windows low-level hook，可能受安全软件、权限或系统策略影响。
- 睡眠请求可能被系统电源策略、外接设备或其他应用阻止。
- 当前版本没有安装器、密码解锁或远程解锁。
- 全局快捷键可能被其他程序占用；占用时主窗口会显示提示。

## 手动验证清单

发布或打包前建议在 Windows 10/11 实机验证：

- `dotnet build` 能成功。
- 主窗口可保存图片来源、时间限制和倒计时显示设置。
- 预设可以保存、应用、覆盖和删除。
- `Ctrl + Alt + B` 可以唤起窗口并开始 3 秒确认倒计时。
- 关闭主窗口后托盘图标仍然可用，托盘“退出”可以完整结束程序。
- 开机自启选项可以正确写入和移除当前用户启动项。
- 确认倒计时期间点击取消和按 `Esc` 都能取消。
- 儿童模式覆盖所有显示器。
- 常规键盘和鼠标输入被拦截。
- `Ctrl + Alt + U` 长按 3 秒可以解锁。
- 超时后优先请求系统睡眠。
- 退出或解锁后窗口、计时器和输入钩子被清理，系统恢复正常输入。
